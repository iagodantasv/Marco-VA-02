# Educação Financeira e Ativos Internacionais

Projeto desenvolvido com foco em **letramento financeiro para o público jovem**, abordando conceitos de organização financeira, investimentos básicos e introdução a ativos internacionais.

O objetivo é criar uma plataforma educativa moderna, acessível e responsiva, utilizando boas práticas de desenvolvimento Front-end e estruturação semântica em HTML.

---

# Sobre o Projeto

Este projeto foi idealizado em parceria com os cursos de:

- Administração
- Ciências Contábeis

A proposta é apresentar conteúdos financeiros de forma simples e visual, ajudando jovens a:

- Organizar suas finanças pessoais
- Entender investimentos básicos
- Conhecer ativos internacionais
- Aprender sobre rendimentos atrelados ao dólar
- Comparar diferentes tipos de investimentos

---

# Objetivos do Projeto

- Promover educação financeira acessível
- Desenvolver páginas responsivas e organizadas
- Aplicar boas práticas de HTML semântico
- Trabalhar tabelas comparativas responsivas
- Melhorar a experiência visual do usuário
- Criar um projeto moderno para portfólio acadêmico
